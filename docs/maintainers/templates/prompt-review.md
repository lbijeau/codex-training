# Prompt Review Checklist

- **Clarity**: Does the prompt spell out the goal and constraints?
- **Helpers referenced**: Are the necessary helper functions called out?
- **Context**: Is sufficient context summarized for Codex to continue?
- **Safety/limits**: Are boundaries specified (e.g., no secrets, time limits)?
- **Follow-up**: Is there a clear next prompt or validation step?
