## Quick Tips

1. **Break into stages**: 3-5 stages for complex work
2. **Clear success criteria**: Define "done" for each stage
3. **Use TodoWrite**: Track progress in real-time
4. **Plan before implementing**: Especially for multi-file changes
5. **Iterate on plan**: Adjust as you learn
